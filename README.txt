마린하우스코리아(주) 제조업 분위기 홈페이지 최종본

구성 파일
- index.html : 메인
- about.html : 회사소개 / 연혁 / 개발자료
- products.html : 제품소개
- business.html : 사업영역
- cases.html : 설치사례
- support.html : 고객지원 / 상담신청 폼
- location.html : 오시는 길
- assets/css/style.css : 전체 디자인
- assets/js/main.js : 모바일 메뉴
- assets/images/ : 로고 및 이미지

반영 내용
- 사용자가 제공한 마린하우스코리아(주) 로고 적용
- 기존의 가벼운/네온 위주 분위기를 줄이고, 네이비·화이트·스틸톤 제조업 회사 스타일로 재구성
- 회사소개 PPT 텍스트를 마린하우스코리아 기준으로 재작성
- 기존 회사소개 자료를 마린하우스코리아 기준으로 정리
- 주소 반영: 경기 김포시 양촌읍 삼도공단로63번길 20
- 상담 메일 반영: inbopp3@gmail.com

주의
- 대표 전화번호는 확정 번호가 필요해 현재 페이지에는 노출하지 않았습니다. 번호가 확정되면 footer/support/location에 추가하면 됩니다.
- support.html의 상담폼은 FormSubmit 방식입니다. 최초 사용 시 inbopp3@gmail.com에서 인증 메일 확인이 필요할 수 있습니다.
