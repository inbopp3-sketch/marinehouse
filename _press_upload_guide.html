보도자료 등록 가이드

이 파일은 내부 작업용입니다. 홈페이지 메뉴와 사이트맵에는 연결하지 않았습니다.

현재 방식

현재 홈페이지는 GitHub Pages 정적 사이트라서 홈페이지 화면에서 로그인 후 글을 쓰는 게시판 방식은 아닙니다. 보도자료는 HTML 파일을 추가해서 올리는 방식으로 운영합니다.

새 보도자료 올리는 순서

  1. _press_template.html 파일을 복사합니다.
  2. 파일명을 영문 소문자와 하이픈으로 저장합니다. 예: press/new-machine-launch.html
  3. 파일 안의 제목, 날짜, 요약, 본문을 새 보도자료 내용으로 바꿉니다.
  4. press.html에 새 보도자료 카드 1개를 추가합니다.
  5. sitemap.xml에 새 보도자료 URL을 추가합니다.
  6. GitHub에서 변경 파일을 업로드하고 Commit 합니다.

고객에게 보이는 부분

  • 고객은 press.html 보도자료 목록만 봅니다.
  • 이 가이드 파일은 메뉴에 노출되지 않습니다.
  • 템플릿 파일도 검색 수집되지 않도록 설정했습니다.

네이버 검색 반영용 체크

  • 보도자료마다 독립 URL을 사용합니다.
  • 제목과 요약을 title, description에 반영합니다.
  • sitemap.xml에 새 URL을 추가합니다.
  • 네이버 서치어드바이저에서 사이트맵을 제출합니다.
